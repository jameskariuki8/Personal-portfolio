Road to become a web developer.

Web development is categorised in to two sections, frontend and backend.To become a frontend developer, you need to learn HTML, CSS and Javascript. Backend Comprises of Python, Database and an interface framework like Django, Node.js or React.js.

How to learn programming

Make the fundamentals right. After understanding the fundamentals, start doing project either from youtube or any other platform you find relevant. As time goes by you will realize that you can code by yourself. Bring your originating ideas into reality using coding.

Backend in one month

Start learning a framework like django(most popular). After you have finished django choose a database query language like MySQL for relational database. Remember its necessary for you to learn frontend before learning backend.

How to maintain 80% efficiency during coding.

Use extentions provided by the code editor. Visual Studio Code being the most popular code editor, has alot of extensions like livesever which makes coding easy. Use AI tools like chartgpt to generate or debug codes.

Frameworks makes tasks easy.

In frontend development frameworks like bootsrap and SCSS makes work a bit easier becouse it is a combination of both javascript and CSS. Utilizing this frameworks makes you as a developer to complete a task within a short period of time.
